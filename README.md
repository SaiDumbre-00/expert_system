🩺 Medical Expert System (Tkinter & ReportLab)
An interactive Medical Expert System built using Python, Tkinter, and ReportLab. This project allows users to input patient symptoms and receive a probable diagnosis along with advice, making it a simple educational tool for understanding basic rule-based expert systems.

🚀 Features
🎯 Enter patient name
🧾 Select symptoms from a predefined list
⚡ Real-time diagnosis with confidence levels
📝 Advice for each possible disease
📄 Export diagnosis report as a PDF
🧼 Clear selections and restart anytime

🎮 Controls

Action	Input
Enter Patient Name	Text Field
Select Symptoms	Click Buttons
Diagnose	Click Diagnose Button
Export PDF	Click Export PDF Button
Clear	Restart App or Clear Selections

🧠 How It Works
The system uses a rule-based knowledge base:

Matches selected symptoms with predefined disease rules
Calculates confidence level based on symptom overlap
Displays all possible diseases sorted by confidence
Provides medical advice for each matched disease
